# Биржа исходов

Мобильная HTML5-игра на виртуальные очки.

## Играть

Открыть игру сразу в браузере:

https://raw.githack.com/Phenolemox/Main/main/index.html

## Правила

- Стартовый баланс: 500 очков
- Исход 1.5×: шанс победы 70%
- Исход 2.2×: шанс победы 30%
- При победе начисляется чистая прибыль по коэффициенту
- При проигрыше ставка списывается
- Прогресс сохраняется в браузере через localStorage
